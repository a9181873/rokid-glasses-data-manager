# 眼鏡檔案站使用說明

繁體中文｜[English](USER_GUIDE.en.md)｜[日本語](USER_GUIDE.ja.md)

## 先知道三件事

1. **手機不需另裝檔案管理 App。** Safari、Chrome 或其他現代瀏覽器即可。
2. **不會預先同步相簿。** 清單與縮圖按需讀取；只有你按下下載時，手機或電腦才會建立副本。
3. **敏感檔案優先使用 USB。** Wi‑Fi 模式請只用自己的手機熱點或可信任網路。

## 首次安裝

### 方法 A：Hi Rokid Toolbox

1. 更新 Hi Rokid 與眼鏡 YodaOS-Sprite。
2. 在 Hi Rokid 開啟 Toolbox。
3. 選擇交付的 `GlassesFiles.apk` 並安裝到眼鏡。
4. 戴上眼鏡，在 App 清單末端尋找「眼鏡檔案站」。

不同地區的 Hi Rokid 介面可能略有差異。若沒有 Toolbox，使用方法 B。

### 方法 B：ADB 除錯線

1. 準備具有資料接點的 Rokid 除錯線；盒內充電線不一定能傳資料。
2. 在 Hi Rokid／眼鏡設定中開啟開發者模式與 ADB debugging。
3. 連接電腦並確認：

```bash
adb devices -l
```

4. 安裝 APK：

```bash
adb install -r GlassesFiles.apk
```

## 首次授權

1. 開啟「眼鏡檔案站」。
2. 選「允許檔案管理」。
3. 在 Android 設定頁允許「所有檔案存取」。
4. 返回 App。授權會保留，不必每次重做。

App 雖取得檔案管理器權限，實際程式仍只管理 `DCIM/Camera`、`DCIM/album`、`Pictures`、`Movies` 等媒體白名單，不瀏覽系統檔或其他 App 私人資料。

## 眼鏡端操作

| 動作 | 功能 |
|---|---|
| 向前／向後滑 | 上一項／下一項 |
| 點按 | 開啟目前項目或執行按鈕 |
| 雙擊 | 返回上一層 |
| 長按 | 開啟項目操作（韌體若保留給 AI，改用點按後的操作頁） |
| 實體返回／DPAD | 同 Android 標準返回與方向鍵 |

首頁提供：

- 今日拍攝
- 所有相片與影片
- 僅相片／僅影片
- 已保護
- 我的最愛
- 大檔案
- 重複檔案
- 垃圾桶
- USB 電腦管理
- Wi‑Fi 手機管理
- 儲存授權與空間資訊

### 保護重要檔案

在檔案操作頁選「設為保護」。受保護項目不能重新命名或移入垃圾桶，必須先手動解除保護。標記只存在眼鏡 App 私有設定，不修改原始媒體內容。

### 刪除與復原

- 一般「刪除」只會移到可復原垃圾桶。
- 垃圾桶可還原原位置；若名稱已被占用，App 會停止並要求先處理衝突。

## 手機瀏覽器管理與遙控

1. 開啟自己的手機熱點。
2. 讓眼鏡連上該熱點。
3. 在眼鏡首頁選「Wi‑Fi 手機管理」。
4. 用手機相機掃描眼鏡顯示的離線 QR Code；若相機無法辨識，再手動輸入例如 `http://192.168.43.20:8765`。
5. 輸入當次六位配對碼。

分享與手機操作期間，眼鏡螢幕會保持常亮，方便同步查看遙控結果；停止分享或閒置 10 分鐘自動停止後會恢復一般休眠。QR Code 只在眼鏡本機產生，不會傳送網址到外部服務。

配對後可：

- 依今天、指定日期、相片、影片、保護、我的最愛、大檔與重複群組篩選。
- 直接串流預覽；影片支援 Range，拖曳或斷線後不需重傳整段。
- 對單一項目切換保護／最愛、重新命名、移入垃圾桶或下載。
- 由「眼鏡遙控」的上一個、下一個、開啟、返回四個大按鈕操作眼鏡畫面。

完成後在眼鏡選「停止分享」。服務若 10 分鐘沒有操作，也會自動關閉。

## USB 電腦管理

1. 在眼鏡首頁選「USB 電腦管理」。
2. 連接除錯線並執行：

```bash
adb forward --no-rebind tcp:8765 tcp:8765
```

3. 電腦瀏覽器開啟 `http://127.0.0.1:8765`。
4. 輸入眼鏡顯示的配對碼。
5. 完成後停止分享並移除轉送：

```bash
adb forward --remove tcp:8765
```

## 重複檔與大檔整理

- 大檔案依檔案大小直接排序，不需先讀完整內容。
- 重複檔掃描先以大小分組，再以小型固定緩衝區串流計算 SHA‑256；不會把影片載入 RAM。
- 預設只在充電時掃描。未充電時仍可手動開始，但眼鏡可能較快耗電或升溫。
- 掃描可以取消；取消不會修改任何原檔。
- 同一雜湊群組仍由使用者決定保留哪一份，App 不自動刪除。

## 常見問題

### 手機連不上網址

- 確認手機與眼鏡在同一個私人熱點／Wi‑Fi。
- 確認使用眼鏡當次顯示的 IP，而不是舊書籤。
- 關閉手機 VPN 或會阻擋區域網路的安全工具後重試。
- 停止再重新開啟分享，配對碼也會隨之更新。

### 看得到檔案但不能改名或刪除

目前使用的是 MediaStore 唯讀備援。回到首頁選「允許檔案管理」，並確認 Android 設定中的「所有檔案存取」已開啟。

### 找不到剛拍的照片

返回首頁再重新整理。如果仍找不到，依 [真機驗收清單](DEVICE_CHECK.md) 確認原廠相機的實際路徑與 MediaStore 資料。

### 影片在瀏覽器不能播放

這通常是瀏覽器不支援影片編碼，不代表下載檔損壞。先下載原檔，再用支援該編碼的播放器開啟。

### App 被系統停止

眼鏡電池低、折疊休眠或 YodaOS 省電機制可能停止前景服務。重新戴上眼鏡並手動再啟動分享即可；App 不會為了常駐而設定開機自啟。
