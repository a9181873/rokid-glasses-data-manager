# プライバシーについて

[繁體中文](PRIVACY.md)｜[English](PRIVACY.en.md)｜日本語

眼鏡ファイルステーションは、オフラインかつローカル優先で設計されています。

## アプリが行わないこと

- アカウントを作成しません。
- 開発者、Rokid、サードパーティのクラウドサービスへ接続しません。
- 広告、アクセス解析、クラッシュ報告、プッシュ通知、ソーシャル機能、トラッキング SDK を含みません。
- Android の位置情報権限を要求せず、現在位置、連絡先、マイク、カメラ、通話情報、広告識別子を読み取りません。
- 写真、動画、ファイル名を、スマートフォン、パソコン、インターネットへ自動同期しません。

## アプリがローカルで処理するデータ

- 利用者が許可したフォルダ内にある、写真と動画のメタデータ、サムネイル、ストリーミング内容。
- 「保護済み」「お気に入り」、ゴミ箱からの復元先、重複ファイルのインデックス、必要なセキュリティセッション状態。これらのデータはアプリ内のローカル領域だけに保存され、元のメディアへ書き戻されません。

## ブラウザ共有

利用者が眼鏡のフォアグラウンド画面で USB または Wi‑Fi 共有を明示的に起動した場合に限り、アプリは接続の待ち受けを開始します。起動するたびに新しいペアリングコードとセッショントークンを使用し、共有の停止後またはタイムアウト後に無効になります。

ブラウザでプレビューまたはダウンロードすると、選択したメディアがそのスマートフォン／パソコンへ送信されます。その後、スマートフォンやブラウザがコピーを保存、キャッシュ、バックアップする方法は、本アプリの管理対象外です。

Wi‑Fi 版は現在、ローカルネットワーク上の HTTP を使用するため、同じネットワーク上での盗聴を防止できません。機密性の高いデータには USB モードを使用してください。詳細は [SECURITY.ja.md](SECURITY.ja.md)を参照してください。
